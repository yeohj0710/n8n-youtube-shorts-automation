함께 첨부한 레퍼런스(1개 또는 여러 개)를 보고 n8n 로컬 소재 대기열에 추가해줘.

중요 규칙:
- n8n 실행하지 마.
- 워크플로우 실행하지 마.
- YouTube 업로드하지 마.
- 이미지의 메인 제목/후킹 문구는 최대한 그대로 유지해.
- 제목을 새 주제로 바꾸지 마. 어그로성, 궁금증, 강한 훅은 살려.
- 오타나 너무 어색한 표현만 최소 수정해.
- 내용은 그대로 베끼지 말고 살짝 재구성해.
- 순서를 바꾸거나, TOP 순위로 정리하거나, 표현을 조금 다듬어도 됨.
- 질병 치료/예방/수명 보장처럼 위험한 표현은 본문/메모에서 순화해.
- 제목에 있는 강한 훅은 유지해.

여러 이미지가 첨부된 경우:
- 서로 다른 주제면 이미지마다 소재 파일을 1개씩 따로 만들어.
- 같은 주제의 연속 캡처나 보충 이미지면 1개 소재로 합쳐.
- 합칠 때는 가장 강한 메인 제목/후킹 문구를 제목으로 써.
- 여러 소재를 만들 때도 각 소재 제목은 각 이미지의 메인 제목을 최대한 유지해.
- 첨부 순서대로 처리하되, 대기열에 이미 있는 파일은 건드리지 마.

채널 선택:
- 하루건강약사 채널:
  - 약사 느낌의 생활 건강 팁, 영양제, 식습관, 혈당, 수면, 물, 운동 루틴, 일상 건강 정보에 적절하면 사용.
  - 소재 폴더: C:\dev\n8n-youtube-shorts-automation\하루건강약사 소재
- 건강장수비결 채널:
  - 장수, 노화, 중장년/시니어, 오래 사는 습관, 민간요법 느낌, 건강비법, 음식 피하기/챙기기 류에 적절하면 사용.
  - 소재 폴더: C:\dev\n8n-youtube-shorts-automation\건강장수비결 소재
- 애매하면 제목 톤을 보고 더 자연스러운 채널 하나를 골라.

파일 작성 방식:
- 소재 1개당 선택한 채널 소재 폴더 루트에 .md 파일 1개를 만들어.
- 여러 이미지를 각각 소재로 처리하면 .md 파일도 여러 개 만들어.
- 기존 README.txt, 기록 폴더, 사용완료 폴더는 건드리지 마.
- 기존 대기 파일은 삭제/수정하지 마.
- 파일명은 날짜시간 + 핵심 키워드로 만들어도 됨. 파일명은 내부용이라 제목 훅과 달라도 됨.
- 파일 내용은 아래 형식으로 작성해.

형식:

# [이미지의 메인 제목을 최대한 그대로]

Subtitle: [이미지 부제나 영상용 짧은 부제]
Lane: [아래 Lane 중 하나]

Ranks: ('이유'는 필요한 경우에만 추가. 글자가 너무 많을 경우 생성된 이미지 텍스트가 깨짐)
1. [항목] - [짧은 이유]
2. [항목] - [짧은 이유]
3. [항목] - [짧은 이유]
...

Notes:
원본 이미지의 제목 훅은 유지. 소재만 순위형/리스트형으로 재구성.

검증:
- 파일 추가 후, 선택한 채널 폴더의 대기 순서를 확인해줘.
- 최종 답변에는 어느 채널에 넣었는지, 파일 경로, 대기 순서만 짧게 알려줘.
