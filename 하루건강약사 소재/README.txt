하루건강약사 소재 넣는 곳

사용법:
1. 이 폴더에 영상 1개당 .txt 파일 1개를 넣습니다.
2. n8n을 실행하면 먼저 넣은 파일부터 1개씩 골라 업로드합니다.
3. 사용된 파일은 자동으로 사용완료 폴더로 이동합니다.

가능한 파일:
- .txt
- .md
- .json

파일 예시:

아침에 먹으면 혈당이 흔들리기 쉬운 습관 5가지
- 빈속 믹스커피 - 단맛이 먼저 들어옴
- 흰빵만 먹기 - 금방 허기짐
- 주스만 마시기 - 씹는 과정이 없음

주의:
- 이 README.txt 파일은 소재로 사용되지 않습니다.
- 기록 폴더와 사용완료 폴더는 건드리지 않아도 됩니다.
